# Quote categories

The deterministic quote miner uses these category labels:

- `pain_points`
- `day_to_day_struggles`
- `victories`
- `failures`
- `goals`
- `beliefs`
- `desires`
- `objections`
- `trigger_moments`
- `real_customer_language`

These are routing labels, not final truths. Final synthesis should verify and refine them.
